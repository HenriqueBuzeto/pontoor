# Ponto OR — Design System

## Identidade visual

- **Nome**: Ponto OR
- **Paleta**: Preto, Laranja, Branco
- **Direção**: Premium, limpo, corporativo, moderno, alto contraste, legível, responsivo, elegante.

## Tokens de cor (Tailwind)

| Token | Uso | Hex (light) |
|-------|-----|-------------|
| `--ponto-black` | Texto principal, bordas, sidebar | #0a0a0a |
| `--ponto-orange` | Primária, CTAs, links, destaques | #f97316 |
| `--ponto-orange-hover` | Hover em botões primários | #ea580c |
| `--ponto-orange-muted` | Backgrounds sutis, badges | #fff7ed |
| `--ponto-white` | Background principal | #ffffff |
| `--ponto-muted` | Texto secundário | #71717a |
| `--ponto-border` | Bordas | #e4e4e7 |
| `--ponto-success` | Sucesso | #22c55e |
| `--ponto-warning` | Avisos | #eab308 |
| `--ponto-error` | Erro, destaque negativo | #ef4444 |

## Tipografia

- **Font sans**: System UI ou Inter / Geist — títulos e corpo.
- **Font mono**: Para dados numéricos e códigos (matrícula, IDs).
- **Hierarquia**: h1 (2xl), h2 (xl), h3 (lg), body (base), small (sm), caption (xs).

## Estados

- **Hover**: Escurecer ligeiramente ou `ponto-orange-hover` em primários.
- **Focus**: Ring 2px laranja, outline acessível.
- **Disabled**: Opacidade 0.5, cursor not-allowed.
- **Loading**: Skeleton ou spinner laranja.

## Dark mode (opcional)

- Manter laranja como primária; fundo preto/cinza escuro; texto branco/cinza claro; bordas sutis.

## Logo textual

- **Ponto** em preto ou branco (conforme fundo).
- **OR** em laranja, mesmo peso ou bold.
- Fonte sans-serif moderna, sem serifa.
